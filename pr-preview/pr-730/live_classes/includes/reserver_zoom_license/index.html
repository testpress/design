Reserve Zoom License

A Zoom license will be automatically assigned to when you start the class based on availability. However, if needed, you can manually reserve a Zoom license for this class beforehand. After the class ends, the license will become available for future classes, allowing multiple staff users to share a single Zoom license.

You can choose to reserve a license without starting the class immediately or reserve and start the class now.

Once reserved, a license cannot be used to start any other class until it is automatically freed up.
