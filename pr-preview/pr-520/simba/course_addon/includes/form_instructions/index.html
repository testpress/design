Adding a Course Addon

Complete the fields below to enroll a student in an addon course:

  • Addon Courses: Select one or more addon courses for enrollment.
  • Payment Details:
    • Enter the exact amount for the courses selected (partial payments are not allowed).
    • Provide the transaction reference and choose a payment method.
  • Notes: Optionally, add any relevant notes for this transaction.

Ensure all required fields are filled out. Click "Submit" to finalize the addon course enrollment.
