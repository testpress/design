Changing a Student's Batch

Complete the fields below to update the student’s batch in the selected course:

  • Course: Select the course in which the student’s batch needs to be changed.
  • Preferred Batch: Choose the preferred batch from the list of available options.
  • Reason for Batch Change: Provide a valid reason, such as schedule conflicts or academic needs.
  • Conditions:
    • Capacity Check: Confirm that the new batch has available slots.

Click "Change Batch" to finalize the update.
