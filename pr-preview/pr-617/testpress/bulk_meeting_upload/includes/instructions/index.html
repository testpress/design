Instructions & Sample Format

Upload multiple video conference meetings at once using an Excel file. Ensure your column headers and data formats match the sample table below.

Supported file types: Excel file (.xls, .xlsx).

Note: First row in the excel should contain the below headings apart from the optional ones and can be of any order.

Title
Required. The display name for the video conference session that users will see.
Start Date
Required. The date the meeting is scheduled to begin. Must be in YYYY-MM-DD format.
Start Time
Required. The time the meeting is scheduled to start. Must be in 24-hour HH:MM format.
Duration (Minutes)
Required. The total planned length of the meeting in minutes (e.g., 60 for one hour).
Provider
Required. Specify the video conferencing service. Must be exactly zoom or teams (case-insensitive).
Host
Required. The email address of the registered user who will host the meeting. This user must exist in the system.
Batches
Optional. A comma-separated list of existing batch names that should have access to this meeting (e.g., Batch A, Mumbai Batch ). Leave empty if not applicable and it will consider the course level batch for the meeting.
Meeting Placement Path
Optional. Defines where the meeting appears in your course library. Paths are processed case-insensitively and ignore leading/trailing whitespace.
  • Use the greater-than symbol (>) to indicate hierarchy (e.g., Course Name > Chapter Name).
  • Use a semicolon (;) to place the same meeting in multiple locations (e.g., Course A > Chap 1; Course B > Chap 2).
  • Important: The system validates all paths against your course structure. If any path (or part of a path) is invalid or ambiguous (e.g., duplicate course names), the entire row will fail processing.
Show Recorded Video
Optional. Enter Yes or No. If 'Yes', the recording becomes visible to students after the session. Defaults to 'No' if left blank.
Send Invites
Optional. Enter Yes or No. If 'Yes', automatically sends calendar invites to members of the specified batches (requires 'Batches' column to be filled). Defaults to 'No' if left blank.
Required Watch %
Optional. A number between 0 and 100. Sets the percentage of the recorded video a student must watch for completion. Defaults to 0 if left blank.
