Batches

Batches name will appear here once start date, course, branch and timings are selected.

Batches will be created with the following names, and a suffix from 001 to 999 will be added based on availability.

Creating a new batch

To create a new batch, please fill out the following details:

  • Course: The course that this batch belongs to.
  • Maximum Capacity: The maximum number of students allowed to enroll in this batch.
  • Fee: The total fee for enrolling in this batch.
  • Enrollment Open Date: The date when enrollment for this batch opens.
  • Enrollment Close Date: The date when enrollment for this batch closes.
  • Start Date: The date when the batch begins.
  • End Date: The date when the batch concludes.
  • Status: The current status of the batch (e.g., active, inactive, or completed).
  • Mentor: The mentor assigned to this batch who will guide the enrolled students.

Once all required fields are filled, click the "Create Batch" button to save the new batch.
