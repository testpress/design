Basic Information

  • Branch: Select where the session takes place.
  • Date: The day all sessions will be scheduled.

Quick Fill with Session Code

Use a previously saved session code to autofill the curriculum fields instantly.

Session Curriculum

  • Select the Course associated with the session.
  • Choose the Subject under that course.
  • Pick a Teaching Slot for the specific topic/unit.

Batches

Select one or more batches to which this session applies. You can search and filter the list.

Schedule Sessions

  • Date is common for all sessions you create below.
  • Each session can be of type Regular Teaching, Test or Assessment, or Mentor Session.
  • Time Slot is mandatory for each session.
  • Room and Staff are required for Regular Teaching and Mentor sessions
  • Provide a Testpress Exam ID only if type is Test or Assessment.

Once all required fields are filled out, click “Create Session(s)” to finalize the plan.
