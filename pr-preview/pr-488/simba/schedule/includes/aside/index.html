Creating a Class Session

Please fill out the following fields to schedule a new class session:

  • Date: Choose the date the session will be conducted.
  • Time Slot: Select the appropriate time block for the session.
  • Branch: Select the branch where this session will take place. This filters available staff, rooms, and batches.
  • Session Type: Choose the type of session â€” Regular Teaching, Test, Activity, or Discussion.
  • Subject: Pick the subject. This is required only for regular teaching sessions.
  • Teaching Slot: Select a planned teaching slot. This is required only for regular teaching sessions.
  • Testpress Exam ID: For test-type sessions, enter the corresponding exam ID to link it with the Testpress platform.
  • Batches: Select one or more batches who will attend this session. This is mandatory.
  • Staff: Assign either a Teacher or a Mentor. You must choose only one, not both.
  • Room: Optionally select the room where this session will be held.

Once all required fields are filled out, click the "Create" button to save the new session.
