Changing a Student's Course

Update the student’s course and batch by completing the fields below:

  • Select Current Course: Choose the student’s existing course and batch.
  • Preferred Course and Batch: Select the desired course and batch for the student.
  • Reason: Provide a valid reason for the change.
  • Fee Adjustment: Review the fee difference and notify the student if additional payment is required or a credit note is issued.
  • Payment Details: If any adjustment is needed, provide the payment amount, method, transaction ID, and additional notes for record-keeping.

Click "Approve and Change Course" to apply the update or "Escalate to Branch Manager" for further approval.
