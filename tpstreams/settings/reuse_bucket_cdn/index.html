Reuse bucket and CDN

Copy the following code to use the current organization bucket and CDN in your shell.

    
      org = Organization.objects.get(uuid="")
      org.bucket_name = u""
      org.bucket_secret_token = u""
      org.cdn_id = u""
      org.cdn_url = u""
      org.cloudfront_key_group_id = u""
      org.save()
