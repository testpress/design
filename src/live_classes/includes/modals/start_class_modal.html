Meeting ID will be generated automatically when you start the class. However, if needed, you can manually generate it by reserving a Zoom license for this class beforehand. After the class ends, the license will become available for future classes, allowing multiple staff users to share a single Zoom license.

You can choose to reserve a license without starting the class immediately or reserve and start the class now.

Once reserved, a license cannot be used to start any other class until it is automatically freed up.
