Reuse bucket and CDN

Copy the following code to use the current organization bucket and CDN in your shell.

    
      org = Organization.objects.get(uuid="{{ user.current_organization.uuid }}")
      org.bucket_name = u"{{ user.current_organization.bucket_name }}"
      org.bucket_secret_token = u"{{ user.current_organization.bucket_secret_token }}"
      org.cdn_id = u"{{ user.current_organization.cdn_id }}"
      org.cdn_url = u"{{ user.current_organization.cdn_url }}"
      org.cloudfront_key_group_id = u"{{ user.current_organization.cloudfront_key_group_id }}"
      org.save()
