Testing Tools

After you answer a question, select Next. You may still change the answer after selecting Next.
Selecting Back will take you to the previous screen or question.
Selecting Pause Test will pause the test. When the pop-up opens, selecting Continue will pause the test, or selecting Return will return you to the test without pausing it. When you pause a test, you may return to it at any time during the period your test is activated. The Pause Test button is not available in the actual test.
Selecting Help will bring you to help. From Help you can get information on different topics by selecting one of the tabs on screen.
Selecting Review will allow you to access the Review screen. The Review screen lists all of the questions in their part of the Review section and their status. In Untimed Mode, the Status column shows if a question answer is Correct or Incorrect, or if the question has been Not Answered or Not Encountered. In Timed Mode and in the actual test, the Status column shows if a question has been Answered, Not Answered, or Not Encountered. By selecting and highlighting a question and then selecting Go To Question you can return to that question.
Selecting Guess will mark the question as a guess. You can review items that you have marked after you have completed the test. Your detailed score report will include the percentage of correct guesses. The Guess button is not available in the actual test or in Timed Mode in this practice test.
Selecting Show Solution will display the correct solution. The correct solution will be highlighted in yellow. The button will change to Hide Solution. The Show Solution button is not available in the actual test or in Timed Mode in this practice test.
Selecting Hide Solution will remove the yellow highlighting. The button will change to Show Solution. The Hide Solution button is not available in the actual test or in Timed Mode in this practice test.
Selecting Exit Section will allow you to leave a section and move to the next section. Once you have left a section, you will not be able to return to it. The Exit Section button is not available in the actual test or in Timed Mode in this practice test.
Select Return to go on.
