Important: Provider Specific API Rate Limits

Please be aware of the following provider specific API rate limits when creating classes in bulk:

  • Zoom: Limited to 100 create or update requests per user per day.
  • MS Teams: Limited to 2,000 meetings per user per month.

To avoid hitting these limits and ensuring operational continuity, we recommend limiting usage of bulk upload classes to a maximum of 30 classes per day. Exceeding these limits may prevent you from creating urgent manual classes later in the day.
