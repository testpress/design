Describe the principles of object-oriented programming.

Object-oriented programming (OOP) is a programming paradigm based on the concept of "objects". The four main principles of OOP are Encapsulation (bundling data and methods), Abstraction (hiding complex details), Inheritance (reusing code from parent classes), and Polymorphism (objects taking multiple forms).
